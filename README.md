# Loan-Analysis-Dashboard-and-Report
Multi-page Power BI loan analysis report with risk segmentation, DAX measures, and geography-wise customer insights.
## Loan Analysis Dashboard

**Tools:** Power BI | DAX | Power Query | Multi-Page Report

### Overview
An interactive multi-page Power BI report built to analyse loan portfolio 
performance, identify high-risk borrowers, and support informed lending 
decisions through data-driven insights.

### Pages Covered
- Executive Dashboard
- Risk Analysis
- Financial Analysis
- Repayment Analysis
- Geography-Wise Analysis
- Customer Analysis

### Key Features
- Advanced DAX measures: Default Rate, Loss Rate, On-Time %, Recovery Amount, Customer Risk Rank
- Risk segmentation visuals: scatter charts, donut charts, trend lines
- High-risk borrower identification by DTI ratio, credit band, and geography
- Interactive slicers, KPI cards, and drill-through visuals
- Data cleaned and modelled using Power Query across multiple tables

### How to View
1. Download the ZIP file
2. Extract it
3. Open the `.pbix` file in Power BI Desktop
